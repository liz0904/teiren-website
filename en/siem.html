Teiren SIEM

Teiren SIEM provides optimized security and log management with improved performance and convenient visualization using Graph DB.
Use the best SIEMs that can address not only integrated log and event management of enterprise systems, but also security, asset management, and compliance management.
Features

Main Features

Dashboard Teiren SIEM dashboard allows you to monitor real-time threats by viewing the total number of logs and threat logs, threat rates, and recent detection threats. Also, Teiren SIEM provides real-time graphs of CPU/Memory/Network usage.
Integrations You can integrate various clouds and systems by entering API keys.
Integration List NCP ,NHN Cloud, AWS, Window Security Log, Jiran Security Office Keeper, etc
Log Management It collects vast amounts of log data from various clouds and systems and integrates the collected log data into a table form on the web. JSON, Web Table + Filtering(for search)
Threat Detection You can set security policies for threat detection. More than 150 out-of-the-box policies are provided, where simple threat detection is possible. You can add policies directly to your environment with advanced settings. Threats detected by the policy can be found on the Threat Alert page.
UEBA User and Entity Behavior Analytics Threats can be detected based on the flow of behavior, not just one anomaly. Visualize and display the flow of user behavior to the threat at a glance when detecting the threat.
Visualization Visualize user behavior at a glance. You can see what flow the detected threat is through and how it relates to the other actions of the user who caused the threat, which allows you to correlate between them. Detect threats through user behavior pattern-based machine learning
Compliance Report Provides compliance certification reports. It automatically maps compliance items that the enterprise is complying with or must comply with, and provides proof of this in a captured version of the report in pdf format. Comliance Report(ISMS-P, CSAP, etc), Summary report and Excel report on logs detected by threats, interlocking status, etc

Why Teiren SIEM?

Advanced threat detection

Using Graph DB, Teiren SIEM can analyze based on the relationship between logs to perform more advanced threat detection.

Interworking between systems enables integrated security management through correlation analysis of the entire system and assets.

Increase work efficiency

In addition to multi-cloud environments, security solutions, systems, and more can be integrated to manage multiple environments simultaneously with just one SIEM.

In addition, compliance reports on security certification can help improve the efficiency of security managers.

Applicable to your company

Teiren SIEM supports Korean clouds(Naver,NHN, KT Cloud, etc) that currently lack support.

Teiren SIEM can be applied not just to SMEs, but also to major information and communication infrastructures in the country.
